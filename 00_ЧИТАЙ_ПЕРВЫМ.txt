🚀 БЫСТРЫЙ СТАРТ (30 секунд):
═══════════════════════════════════════════════════════════════════════════

ВАРИАНТ 1 - САМЫЙ ПРОСТОЙ (только для Windows):
─────────────────────────────────────────────────

1. Извлеките архив CodeEditor_Source.zip
2. Двойной клик на файл: build.bat
3. Ждите 2-5 минут (идёт установка)
4. Готово! Вам будет предложено запустить приложение

⚠️  Требуется:
   • Windows 7+
   • Python 3.8+ (скачайте с python.org если не установлен)
   • Интернет (для установки зависимостей)


ВАРИАНТ 2 - ЕСЛИ БАТНИК НЕ РАБОТАЕТ:
──────────────────────────────────────

На Windows (PowerShell):
  1. Нажмите WIN+X → выберите PowerShell (Admin)
  2. cd C:\путь\к\файлам
  3. powershell -ExecutionPolicy Bypass -File build.ps1

На Windows (Command Prompt):
  1. Откройте cmd.exe
  2. cd C:\путь\к\файлам
  3. pip install -r requirements.txt
  4. pip install pyinstaller
  5. pyinstaller --onefile --windowed --name="CodeEditor" code_editor.py


ВАРИАНТ 3 - ДЛЯ MAC/LINUX:
──────────────────────────

  1. pip install -r requirements.txt
  2. python code_editor.py  (для запуска приложения)

Для создания исполняемого файла нужна компиляция под вашу ОС.

🔧 РЕШЕНИЕ ПРОБЛЕМ:
═══════════════════════════════════════════════════════════════════════════

❌ ПРОБЛЕМА: "Python не найден"
✅ РЕШЕНИЕ:
   1. Скачайте Python с https://www.python.org (версия 3.8+)
   2. При установке ОБЯЗАТЕЛЬНО отметьте "Add Python to PATH"
   3. Перезагрузитесь
   4. Повторите build.bat

❌ ПРОБЛЕМА: "pip не найден"
✅ РЕШЕНИЕ:
   • Это обычно значит что Python не добавлен в PATH
   • Переустановите Python с галочкой "Add Python to PATH"

❌ ПРОБЛЕМА: "Ошибка доступа при запуске батника"
✅ РЕШЕНИЕ:
   1. Откройте PowerShell (от администратора)
   2. Выполните: powershell -ExecutionPolicy Bypass -File build.ps1

❌ ПРОБЛЕМА: "ModuleNotFoundError: No module named 'PyQt6'"
✅ РЕШЕНИЕ:
   pip install --force-reinstall -r requirements.txt

❌ ПРОБЛЕМА: "Приложение запускается медленно"
✅ РЕШЕНИЕ:
   • Это нормально в первый раз (PyQt6 загружается)
   • Со временем будет запускаться быстрее
   • На современных компьютерах запускается за 2-3 секунды

❌ ПРОБЛЕМА: "Антивирус блокирует exe"
✅ РЕШЕНИЕ:
   • Это нормально для свежесозданных exe
   • Добавьте приложение в исключения антивируса
   • Или используйте вариант запуска через Python: python code_editor.py
