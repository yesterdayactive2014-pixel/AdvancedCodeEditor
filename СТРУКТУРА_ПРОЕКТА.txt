САМЫЙ ПРОСТОЙ СПОСОБ (для Windows):
────────────────────────────────────

1. Откройте папку с файлами
2. Двойной клик на файл: build.bat
3. Ждите 2-5 минут (идёт установка)
4. Когда батник спросит "Запустить прямо сейчас?" - напишите y
5. Готово! Приложение откроется


ЕСЛИ БАТНИК НЕ РАБОТАЕТ:
─────────────────────────

1. Откройте PowerShell (WIN+X → PowerShell)
2. Перейдите в папку: cd C:\путь\к\файлам
3. Выполните: powershell -ExecutionPolicy Bypass -File build.ps1

🔧 РЕШЕНИЕ ПРОБЛЕМ:
═══════════════════════════════════════════════════════════════════════════

❌ "Python не найден"
   → Скачайте Python с python.org (версия 3.8+)
   → При установке отметьте "Add Python to PATH"
   → Перезагрузитесь

❌ "Батник не работает"
   → Используйте build.ps1 вместо батника
   → Или выполните вручную команды из build.bat

❌ "PyQt6 не установлен"
   → pip install --force-reinstall -r requirements.txt

❌ "PyInstaller не найден"
   → pip install pyinstaller

❌ "Размер exe слишком большой"
   → Это нормально (250-400 MB), так как включает Python
   → Можно сжать в zip (будет ~80-120 MB)

❌ "Приложение медленно запускается"
   → Это нормально в первый раз (PyQt6 загружается)
   → На втором запуске будет быстрее
